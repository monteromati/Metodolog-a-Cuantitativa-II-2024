Evaluaci√≥n de los Supuestos de la Regresi√≥n Log√≠stica

Es importante verificar que los supuestos de la regresi√≥n log√≠stica se cumplen para garantizar la validez de las estimaciones del modelo.

  • Variable dependiente categ√≥rica: La variable que estamos tratando de predecir (deserta) es binaria, lo que cumple con el primer supuesto.
  • Independencia de las observaciones: Este supuesto no se eval√∫a estad√≠sticamente, pero es importante asegurarse de que cada observaci√≥n (estudiante) es independiente de las dem√°s.
  • Supuesto de Linealidad: implica que la relaci√≥n entre el logit de la variable dependiente y cada variable independiente es lineal. Verificamos este supuesto usando el Test Box-Tidwell. Si el resultado del test es no significativo, no se rechaza la hip√≥tesis nula de linealidad, lo que indica que el supuesto de linealidad se cumple.
logodds <- m2$linear.predictors
boxTidwell(logodds ~ mat2021_1anio$edad_alu)
